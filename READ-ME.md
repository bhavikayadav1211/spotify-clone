# Spotify Clone 🎵

A responsive Spotify Web Player clone built using HTML and CSS.
This project recreates the modern Spotify interface with a clean UI, sidebar navigation, music cards, and a functional music player layout.

## 🚀 Features

* Spotify-inspired responsive UI
* Sidebar navigation menu
* Sticky navigation bar
* Music cards and playlists section
* Custom music player controls
* Modern dark theme design
* Responsive layout for different screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts (Montserrat)
